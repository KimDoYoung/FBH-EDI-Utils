# EdiRename

## 개요

EDI문서들이 이름을 바꾼다.  문서명 19598-1ecc9905-f2a9-496f-a7c0-33e72dbb80b8.xlsx 을 210-invoiceNo.xlsx로 바꾼다.
이때 210과 invoiceNo는 엑셀파일의 내용을 읽어서 판단한다.

## 기능
1. 선택된 폴더의 엑셀을 읽어서 어떤 문서인지 판단. 즉 210, 850 등 판단
2. 선택된 폴더 하위에 문서번호로 폴더를 만들고 그 폴더 하위에 이름을 바꾼 파일을 copy한다.
3. 동작 (순서대로 클릭)
   - 폴더선택 버튼 : 이름을 바꿀 엑셀파일들이 있는 폴더지정 
   - 리스트만들기 버튼 : 파일들 목록을 왼쪽 리스트로 가져옴
   - 이름바꾸기버튼 : 왼쪽 목록의 파일들의 이름을 해당 폴더 하위에 문서번호 폴더를 만들고 그 하위에 **이름 바꾸는 규칙**에 따라 만듬


## 이름 바꾸는 규칙

 - 210: 210_날짜(B7)_PO#(B4).xlsx
 - 846: 846_날짜(B5)_PO#(B4).xlsx
 - 850: 850_날짜(B5)_PO#(B4).xlsx
 - 945: 945_날짜(B5)_PO#(B8).xlsx
