# EdiDiff

## 개요

1. 850을 중심으로 945에서 품목을 찾아서 갯수가 맞는지 체크한다.
2. 체크한 항목들을 보여주는 excel을 만든다.
3. 945에서는 po + item no로 찾는다.
4. 찾은 결과는 1) 발견못함 2)일치 3) 불일치 3개로 보여준다.
